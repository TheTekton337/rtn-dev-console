# rtn-dev-console

Cross platform terminal Fabric component for react-native using SwiftSH/SwiftTerm (WIP) for iOS and Termux (TODO) for android.

## Installation

```sh
npm install rtn-dev-console
```

## Usage

```js
import { RtnSshTerminalView } from 'rtn-dev-console';

// ...

<RtnSshTerminalView />;
```
